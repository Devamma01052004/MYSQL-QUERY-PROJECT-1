# MYSQL-QUERY-PROJECT-1

## PROJECT OVERVIEW  
THIS PROJECT CONTAINS A COLLECTION OF MYSQL QUERIES PERFORMED ON THE ZEPTO  
PRODUCT DATASET. THE PROJECT FOCUSES ON PRACTICAL SQL USAGE INCLUDING  
FILTERING, AGGREGATION, SUBQUERIES, VIEWS, STORED PROCEDURES, FUNCTIONS,  
TRIGGERS, AND REPORT GENERATION.

---

## DATASET INFORMATION  
DATASET NAME: ZEPTO_V2.CSV  

THE DATASET CONTAINS THE FOLLOWING COLUMNS:
- CATEGORY  
- NAME  
- MRP  
- DISCOUNTPERCENT  
- DISCOUNTEDSELLINGPRICE  
- AVAILABLEQUANTITY  
- WEIGHTINGMS  
- OUTOFSTOCK
